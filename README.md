# Portfólio Pessoal — Giordano Bruno

Portfólio profissional desenvolvido para apresentar projetos, experiência e competências em desenvolvimento de software, infraestrutura e segurança.

## Sobre o projeto

- Aplicação SPA com foco em performance, legibilidade e manutenção.
- Estrutura modular para facilitar evolução contínua do conteúdo.
- Experiência responsiva para desktop, tablet e mobile.

## Boas práticas aplicadas

- Componentização por responsabilidade em `src/components`.
- Separação entre dados e interface em `src/data/portfolioData.js`.
- Reutilização de animações com hook dedicado em `src/hooks/useFadeInUp.js`.
- Organização por seções para escalabilidade de conteúdo e manutenção.
- Estilo consistente com utilitários do Tailwind CSS.
- Build de produção validado para garantir qualidade de entrega.

## Ferramentas e bibliotecas

### Base de desenvolvimento

- React
- Vite
- JavaScript (ESM)

### UI e experiência

- Tailwind CSS
- Framer Motion

### Qualidade e build

- ESLint
- npm scripts (`dev`, `build`, `preview`)
